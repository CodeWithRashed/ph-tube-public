Discuss the scope of var, let, and const??

Answer

  • Var Variable: var is a functional scope variable. We can access the variable anywhere inside the function.
  • Let Variable: let is a block-scope variable; let is only accessible within the block where it is defined. We cannot access them outside of their block. Which starts and ends with "{}" curly braces.
  • Const Variable: const is also the same as let. but we cannot reassign the value of a variable declared with const.
